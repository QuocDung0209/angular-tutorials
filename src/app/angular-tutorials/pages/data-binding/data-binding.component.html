Angular Component

Component là khối xây dựng chính của Angular. Mỗi component tạo ra một thành phần nhỏ cho giao diện ứng dụng web như button, sidebar, header, footer,...

Angular component là một JavaScript class và được định nghĩa bằng @Component decorator. @Component Decorator là một function để chỉ định một class là Component với Angular và để cung cấp view (template) và metadata cho component.

Một Component bao gồm 3 khối xây dựng chính:
  • Template (View): là file .html chứa các HTML element để định nghĩa layout và content cho View. Không có template thì sẽ không có gì được hiển thị (Angular sẽ không render bất cứ thứ gì ra DOM).
    • Ngoài HTML emelent, chúng ta còn có thể thêm các Angular Directive, Angular Pipes và các Angular Component khác vào template.
    • Data cho template sẽ được lấy từ component class. Angular sẽ dụng kỹ thuật Data Binding để giữ cho template và component class đồng bộ với nhau.
    • Template có thể sử dụng Event Binding hoặc Two Way Data Binding để thông báo cho component class khi user tác động lên View (template).
  • Class: Cung cấp data và logic cho View (template). Nó chứa JavaScript code liên kết với Template (View).
    • Chúng ta sử dụng TypeScript để tạo class nhưng chúng ta cũng có thể sử dụng JavaScript trực tiếp trong class.
    • Class chứa các property và method. Các property của class có thể bind đến view sử dụng Data Binding.
  • Metadata: Cung cấp các thông tin thêm cho component.
    Các metadata property quan trong của Component:
    • selector: Chỉ định CSS selector đại diện cho component sẽ được sử dụng trong Template của các component khác. Angular sẽ dựa vào CSS selector này để render component.
    • providers: cung cấp các service mà component sẽ sử dụng.
    • styleUrls: Chỉ định một file sẽ cung cấp CSS styles cho component.
    • templateUrls: Chỉ định file .html sẽ cung cấp HTML template để định nghĩa View cho component.

Chúng ta có 2 cách để tạo ra một component

  1. Dùng Angular CLI để tạo và thêm component vào root module:
    ng g c component-name
  2. Tạo Component thủ công theo các bước sau:
    1. Tạo component file - name.component.ts
    2. Import Component từ angular core - import { Component, OnInit } from '@angular/core';
    3. Tạo component class và export nó - export class DataBindingComponent {}
    4. Thêm @Component decorator bên trên class - @Component({})
    5. Thêm các metadata property vào @Component decorator - selector - templateUrl - styleUrls
    6. Tạo template - file .component.html hoặc inline template trong file .component.ts
    7. Tạo CSS styles - file .component.scss
    8. Đăng ký component trong declarations của Root module - declarations: [NameComponent]
